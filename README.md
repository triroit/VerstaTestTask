# VerstaTestTask

## Запуск проекта
1. Во вкладке Code скачать zip-архив проекта.
2. Распаковать архив в удобную папку.
3. Запустить .sln файл.
4. Запустить проект.

## О проекте
Приложение приемки заказа на основе ASP.Net Core MVC (.Net 8). Использованная база данных - SQLite. ORM - Entity Framework Core.
Функционал:
- Просмотр всех заказов.
- Добавление нового заказа.
- Поиск по заказам.
- Просмотр деталей заказа (по клику на строку таблицы).
